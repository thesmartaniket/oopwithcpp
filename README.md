<h1>Use It</h1>

Clone the project-- <br />
`git clone https://github.com/thesmartaniket/oopwcpp`

Directory-- <br />
`cd oopwcpp`<br/>
`ls`<br/>

Then `cd <folder-name>` to move to the required directory.<br/>

Use `g++ <file-name>` to compile it.<br/>
Then use `./a.out` to run the compiled files.
